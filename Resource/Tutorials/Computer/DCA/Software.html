Software
(Q). What is software?

Answer -A Software is the program that enables a computer to perform a specific task as opposed to the physical components of the system (hardware). Software is the language of a computer. And like human language, there are many different computer languages.

Type of Software

Computer Software falls under 2 basic categories

  • System Software
  • Application Software

System Software : System software is sets of programs, responsible for running the computer, controlling various operations of computer system and management of computer resources.

In the other way we can say it's help human to communicate with hardware

Example : Windows, Unix, Linux, Mac, Android, etc...

Application Software : Application software provide us extra futures of work. Application software run with the help of Operating System.

Example : MS Office, Antivirus, Crome, Mozila etc...
