Computer Operating System

Computer Operating System
(Q). What is operating system?

Answer -An operating system is the most important software that run on a computer. It is a interface between user and computer hardware. Operating system take input from user as a GUI (Graphical User Interface) by input devices and send desire result to user on output devices.

Example of Operating System

  • Linux
  • Windows
  • Mac
  • Dos

Linux Operating System : Linux is freely distributed open source operating system that runs on a number of hardware platforms.

Windows Operating System : Microsoft Windows is a family of operating system for personal and business computers. Windows dominates the personal computer world, offering a graphical user interface (GUI), virtual memory management, multitasking, and support for many peripheral devices.

Mac Operating System: Mac operating system is the official name of the Apple Macintosh operating system Mac OS features a graphical user interface(GUI) that utilizes windows, icons, and all applications that run on a Macintosh computer have similar user interface

Basics of Windows

Windows : Windows is series of operating systems developed by Microsoft. Each version of Windows includes a Graphical User interface, with a desktop that allows users to view files and folders in Windows.

Finding Files and Folders in Windows

We can easly search precreated files and folder on Window

  • Click on Start button
  • Click on search box
  • Enter File/Folder name
  • Press Enter
  • All files and folder with search name will show
Finding Files by File Types

Every file has their own extension which shows after the last period in a file name.

MS office products extention are

  • Word - .doc
  • Excel - .xls
  • Power Point - .ppt

How to search it in Windows

  • Go to Start
  • Click on search button
  • Enter File extension
  • All file with that extention will show.
Control Panel

Control Panel is a part of Microsoft Windows where user can control or change the feature of Hardware and Software

For Example : Setting for the sound, mouse, keyboard, display, network, datetime etc..

How to open Control panel ?

Answer - Click on Start Button On Right hand side you can find Control Panel

Add and Remove / Uninstall a Program

You can find "Uninstall a Program or Add and Remove" Option on control panel where you cand delete or remove application.

Following are the steps..

  • Open Control Panel
  • Click on "Uninstall a Program"
  • Select program
  • Click on Uninstall
Personalize

Personalize Option is use to change your computer Background image, Color, Lock Screen, Theme, Font, Start Menu, Taskbar. It can be accessed from Desktop.

  • Right Click on Desktop
  • Personalize
Date and Time

Date and Time option used to set the calender date, clock time and time zone. Clock is set directly in the system's BIOS.
