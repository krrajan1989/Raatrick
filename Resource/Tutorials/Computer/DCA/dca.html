Computer

Computer
(Q). What is Computer?

Answer -A computer is a machine or device that performs processes, calculations and operations based on instructions provided by a software or hardware program. It is designed to execute applications and provides a variety of solutions by combining integrated hardware and software components.

Basic Operations of Computer

A Computer is a device for inputting raw data. The computer then processes it and gives the user the desired output.

INPUT PROCESSING OUTPUT

What a computer can do.

  • Accept Data
  • Store Data
  • Process Data as desired
  • Retrieve the stored data as and when required
  • Print the result in desired format

Hardware

A Computer system comprises hardware and software. The physical components of the computer which are located in the cabinet are called Hardware.

Software

Software is the program that enables a computer to perform a specific task, as opposed to the physical components of the system.

Computer Hardware Parts

Hardware refers to the physical parts of the computer system.

  • CPU (Central Processing Unit)
  • RAM
  • Motherboard
  • Monitor
  • Keyboard
  • Scanner
  • Printer
  • Mouse
  • Joystick
Peripheral Devices
(Q). What is Peripheral Device?

Answer - Peripheral Devices are connected to the computer externally. These devices are used for performing some specific function.

Peripheral devices are as below

  • Input Devices
  • Output Devices
  • Other Devices
Input Devices
(Q). What is Input Device?

Answer - Input Devices accept data and instructions from the user and send to computer.

Input devices are:

  • Keyboard
  • Mouse
  • Scanner
  • Microphone
  • Joystick
Output Devices
(Q). What is Output Device?

Answer - Anoutput devices are any peripheral that receives data from a computer, usually for display, projection, or physical reproduction.

Output devices are:

  • Monitor
  • Projector
  • Printer
  • Speaker
  • Headphone

Files and Folders

All data and programs in a computer are stored in files and folders.

A New folder is created for each program we install on the hard disk. We can create additional folders to organize our data

Files are placed into each folder, just as we might place individual pieces of paper into file folder.

Folder Creation

Right click on the desktop "New" Option Click on "Folder" to crate a folder

Rename the Folder

Right click on Folder "Rename" Option

Shortccut Key = F2

Delete the Folder

Right click on Folder "Delete" Option

Shortccut Key = Del (Delete)

File Creation

Right click on the desktop "New" Option Click on "Text Document" to crate a file

Rename the File

Right click on File "Rename" Option

Shortccut Key = F2

Delete the File

Right click on File "Delete" Option

Shortccut Key = Del (Delete)
